## ğŸ¯ Quick Tips

### Navigation
- **Zoom**: Mouse wheel or use +/- buttons (bottom right)
- **Pan**: Hold **Shift** and drag, or use middle mouse button
- **Reset view**: Click the maximize icon (bottom right)

### Editing Elements
1. Switch to "Select" mode
2. Click any element on the map
3. Edit properties in the right panel
4. Delete with the trash icon or Delete key

### Team Tokens
- Show team positions with large colored circles
- Automatically uses team colors
- Great for showing initial positions

### Player Tokens
- Show individual player positions
- Uses player's team color
- Shows first 3 letters of player name

### Areas
- Highlight control points, danger zones, or rally points
- Choose rectangle or circle
- Adjust opacity for layering

### Arrows
- Click once for start point
- Click again for end point
- Perfect for showing attack routes or movement paths

## â“ Troubleshooting

**Map won't upload?**
- Check file size (keep under 5MB for best results)
- Use common formats: JPG, PNG, WebP

**Can't see my changes?**
- Wait 0.5 seconds for auto-save
- Check browser localStorage isn't disabled

**Elements won't move?**
- Make sure you're in "Select" mode
- Click element first, then drag

**Lost my plan?**
- Always export important plans as JSON backup
- Check if you're in the right browser (data is per-browser)
